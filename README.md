# vvvjs-landing

Simple Coming Soon landing page for VVV Jewelry Shop.

## Features

- Responsive design for mobile and desktop
- Static HTML with Tailwind CSS (CDN)
- Brand-compliant colors and logo
- Dynamic year in footer

## Deployment

This site is configured for GitHub Pages.

1. Push changes to `main` branch
2. GitHub Pages will auto-deploy

## Customization

Edit `index.html` to update content. Colors are defined in the Tailwind config script in the `<head>`.
